OVERVIEW

应用场景

围绕药考学员日常学习与生活，设备与润德教育生态协同完成以下核心能力。

  1. 1

    药考知识点答疑，支持专业问答及与大咖智能体通话

    润德 API
  2. 2

    语音音频库播放：音频课、专注/阅读模式背景音乐

    硬件 + 润德后端 + 云端
  3. 3

    直播开播提醒、学习任务提醒及发货/节点/生日/面授预约等智能播报

    硬件 + 润德
  4. 4

    番茄时钟

    硬件
  5. 5

    本地备忘录提醒

    大模型 + 硬件
  6. 6

    拨打微信电话

    大模型 + 菊风 SDK + 硬件 + 润德小程序（定位 / VOIP / 地图）
  7. 7

    绑定润德教育 APP 账号（扫码）

    硬件 + 润德
  8. 8

    APP 与学习搭子 MQTT 联动，HTTP 拉取表情与音频

    硬件 + 润德
  9. 9

    GPS 定位

2. 首次启动

2.1 设备激活

首次点击设备开机启动后，进入设备激活页面。

2.2 账号登录

点击激活并连接网络，待开机动画结束后即进入登录页。请使用润德教育 APP扫码绑定账号；二维码有效期为 1 分钟，超时需刷新。

2.3 开机向导

扫码登录成功后进入开机向导（仅首次开机展示）。

💡
指令接收超时：若用户持续 10 秒无语音输入，系统自动结束录音，进入请求处理并返回「无内容」类回应。
  • 扫码登录后进入引导页；可短按 AI 键或呼叫「小狮」唤醒问答，也可选择跳过向导进入首页
  • 唤醒后展示「在听」表情，并随机播报一条唤醒回应（见表情表 #11）
  • 思考完成后按问题作答，回答结束等待 1 秒后进入开机引导第二步
2.4 进入功能菜单

开机引导第二步：用户短按 AI 按键，进入功能菜单页。

3. 锁屏页

3.1 锁屏页

锁屏页显示「静静地待着」表情。若 30 秒内无任何操作，进入待机循环，从 #4–7 表情中随机切换（打瞌睡、无聊、彩虹、呼噜等）；循环持续 1 分钟后自动进入 #39 休眠表情（3s）。点击设置进入控制中心页面。

锁屏页默认展示表情；30 秒后进入待机循环随机表情，循环 1 分钟结束后展示休眠表情 3s。

3.2 控制栏

控制栏显示功能：控制中心、播放中、消息。

3.2.1 控制中心

模块名称默认显示 / 状态点击交互
4G 网络状态显示 4G 信号格；无网时显示断网图标纯展示
音量显示当前音量等级图标唤起音量调节界面
电量显示实时电量百分比 / 图标点击开启 / 关闭「低电量模式」（详见 5.4 低电量模式）
亮度显示当前亮度等级图标唤起亮度调节滑块
静音模式显示当前开关状态（开 / 关）切换静音模式的开启与关闭

3.2.2 播放中

如果存在播放中内容才显示；播放图标需做动态效果，左滑可删除播放内容。

3.2.3 消息通知

灵动岛中仅显示当天的通知内容；点击可进入查看消息详情，左滑删除消息，删除后在消息页面同步删除。

4.1 AI 问答

点击 AI 问答按钮进入 / 使用唤醒词「小狮」/ 长按 AI 按键，均可进入 AI 问答页。进入后展示「在听」表情，并随机播报一条唤醒回应（见表情表 #11）。

唤醒 < 500ms 识别准确率 > 95% 误唤醒 < 1 次 / 24h 多场景降噪

4.2 消息

由甲方返回后台数据对接。点击消息进入查看收到的消息内容，仅保存 30 天内的消息，超过 30 天不显示。左滑可删除消息，点击消息进入查看详情。

4.3 课程

由甲方返回后台数据对接。点击进入课程列表，再进入小节内容；在小节点击进行音频播放。课程名称、小节名称、时长及播放头像均由甲方提供。

  • 返回：继续播放音频，屏幕熄灭后也继续播放
  • 上一个：切换上一个音频，不存在上一个则自动跳转最后一个
  • 下一个：切换下一个音频，不存在下一个则自动跳转第一个

4.4 智能体切换

左右切换选择智能体，切换后点击返回，使用对应选中智能体的音色。共 8 个（润德提供 API，Token 消耗润德的）：

  • 默认：陪伴闲聊智能体（1 个）
  • 药师考试智能体：3 个（中药 1 个、西药 1 个、法规 1 个）
  • 孩子使用智能体：4 个

4.5 提醒事项

由 AI 创建的提醒事项，支持日常通过 AI 添加提醒内容。点击加号唤醒 AI，AI 主动询问：「你好！你希望小狮帮你提醒什么呢？」。暂无提醒时显示对应占位图。

4.6 微信、番茄时钟

使用乙方原本自带的微信内容，本文件不额外补充。

4.7 设置

  • 4.7.1 账号绑定显示扫码绑定的润德教育 APP 账号，可点击退出账号，退出后回到扫码登录页。
  • 4.7.2 通知提醒控制语音播放和宠物动画的开关。若都关闭，不显示宠物动画和自动语音播报（首页的宠物动画不受此控制）。
  • 4.7.3 关于本机显示产品信息。
  • 4.7.4 恢复出厂点击恢复出厂设置。

5. 其他状态

  • 5.1 充电中显示充电中状态，点击屏幕进入首页
  • 5.2 无网络流量到期时，点击任何功能均无网络链接，需到运营商充值后才可以继续使用
  • 5.3 储存空间不足内存不足时提示
  • 5.4 低电量模式省电模式：保留微信通话与定位，触摸禁止，首页「静静地待着」5s 熄屏，长按 AI 键退出（详见下方说明）

5.4 低电量模式

通过控制中心「电量」入口可手动开启；电量 ≤20% 时「需要充电」表情提示结束后也可进入。开启后进入极端省电状态，仅保留必要通信能力。

4. 系统功能说明

  1. 不充电：深度休眠，断掉 mic；充电时 mic 持续开启
  2. 联网正常状态：触摸、表情清醒状态 + 语音交互、正常交互
  3. 休眠状态：断开表情睡眠状态，连接网络和板子，支持语音 / 触摸 / 按键唤醒，恢复至联网正常状态
  4. 深度休眠状态：屏幕熄灭，断开板子，仅支持按键或时钟闹铃唤醒，唤醒至正常状态
  5. 关机状态：断电，仅支持长按开关机键开机唤醒
  6. 充电休眠状态：屏幕熄灭，不断开网络、板子，支持语音唤醒、按键唤醒、闹铃 / 计时器 / 播报时刻唤醒

休眠计时（第 7 条）

  • 未充电：1 分钟无交互 → 进入休眠状态
  • 未充电：3 分钟无交互 → 进入深度休眠状态
  • 充电中：5 分钟无交互 → 进入充电休眠状态，持续状态，不关机

电池策略（第 8 条）

  • 8.1 电量剩余 2% 自动关机，支持开机后提示电量低后自动关机（支持数百次）
  • 8.2 电量剩余 5%，每次启动播放一次需要充电动画
  • 8.3 充电磁吸时，播放显示一次正在充电动画，显示电量后进入正常状态
  • 8.4 充满电，不显示充满电动画
  • 8.5 低电量模式：可通过控制中心手动开启，或电量 ≤20% 提示后进入，规则详见 5.4 低电量模式
EXPRESSION

表情功能介绍

共 38 种表情及触发规则；其中 5 组触摸互动支持表情复用，同一表情可对应多个触发场景。下表各条触发逻辑均受全局规则约束。

全局规则

  • 全局优先级应用层 > 大模型语音交互 > 动作 / 触摸触发。高优先级进行中时，低优先级不响应。
  • 表情状态优先级系统警告 > 功能交互 > 触摸互动 > 待机状态。高优状态触发时，强制打断并清除低优状态。
    系统警告：电量不足、断网、闹铃 / 消息 / 直播提醒等系统级或紧急播报表情。
    功能交互：用户使用系统功能时关联的表情与交互状态，如 AI 问答、微信通话、看课、消息播报等。
    触摸互动：首页通过触摸传感器触发的表情反馈，分组随机：
    屏幕（非眼/嘴）→ 开心 / 眨眼 / 爱心 / 骄傲 / 星星 / 彩虹（六选一）；
    眼睛区域 → 生气 / 难过 / 疑问茫然 / 错误（四选一）；
    背部 → 鼓励 / 兴奋 / 舒服 / 打瞌睡 / 呼噜（五选一）；
    脚丫 → 痒痒笑 / 鬼脸 / 兴奋 / 无聊（四选一）；
    手部 → 骄傲 / 兴奋 / 开心 / 星星 / 鼓励（五选一）。
    待机状态：无用户操作且无功能使用时自动展示，首页默认「静静地待着」（#3），30s 后进入待机循环（#4–7 随机切换 1 分钟），循环结束后进入休眠表情（#39，3s）。
  • 首页专属触发表情仅在首页（锁屏页 / 待机首页）触发显示。用户进入应用功能页后（听课、打电话、课程播放、消息、设置等），触摸互动表情一律禁用，不因触摸触发任何表情切换。
  • 语音保护在「在听」与「说话」状态下，屏蔽所有非紧急触摸互动，防止打断大模型语音交互。
  • 超时兜底「在听」10s 未识别到说话，自动退出并进入「说话」表情，随机播报一条退出文案（见 #14）；「在思考」超时 10s 自动退出，播放退出提示：“哎呀，小狮的耳朵好像没捕捉到信号~ 先乖乖退下，等你呼唤哦！”
  • 断网降级所有联网大模型语音输出，必须配置本地兜底音效，断网时自动降级播放本地音效。
